# Edi Papel de Parede

Site institucional desenvolvido para a empresa **Edi Papel de Parede**, com foco em apresentar os serviços, portfólio de instalações e facilitar o contato com clientes.

## 🌐 Site

**Produção:** https://papeldeparedeedi.netlify.app/

## 📋 Sobre o projeto

O objetivo deste projeto é oferecer uma presença digital moderna para a empresa, destacando:

* Portfólio de instalações realizadas;
* Informações sobre os serviços prestados;
* Formas de contato rápidas;
* Design responsivo para desktop e dispositivos móveis;
* Navegação simples e intuitiva.

## ✨ Funcionalidades

* Landing Page responsiva;
* Galeria de imagens dos trabalhos;
* Botão flutuante do WhatsApp;
* Links para redes sociais;
* Seções de serviços e diferenciais;
* Animações suaves para melhor experiência do usuário;
* Otimização para navegação em dispositivos móveis.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Netlify (Deploy)

## 🚀 Deploy

O projeto é publicado automaticamente através do **Netlify**, realizando novos deploys sempre que alterações são enviadas para o repositório.
